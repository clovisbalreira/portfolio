# Alura Flix V2
📌 **Descrição**
Uma evolução do Alura Flix que permite registrar seus filmes favoritos com pôster e link para o trailer no YouTube.

🚀 **Tecnologias**
- HTML
- CSS
- JavaScript

🎯 **Funcionalidades**
- Adicionar filmes com nome, URL da imagem e link do trailer.
- Exibição dinâmica dos filmes cadastrados na tela.
- Link direto para assistir ao trailer no YouTube.
- Funcionalidade para excluir filmes da lista.

🖥️ **Demonstração**
