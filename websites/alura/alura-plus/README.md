# Alura Plus

📌 Descrição
O projeto Alura Plus é uma landing page moderna e responsiva para um serviço de streaming de educação. A página destaca os benefícios da assinatura do Combo+, que une o Alura+ e o Alura Língua, permitindo que os usuários assistam a cursos em diversos dispositivos e até mesmo offline.

🚀 Tecnologias
- HTML
- CSS
- JavaScript

🎯 Funcionalidades
- Apresentação de planos de assinatura (Combo+ e Alura+).
- Seções informativas sobre a versatilidade de assistir em diferentes telas.
- Destaque para a funcionalidade de download de cursos para assistir offline.
- Lista de dispositivos compatíveis (TV, Computador, Celular).
- Rodapé estruturado com links de navegação e informações legais.
