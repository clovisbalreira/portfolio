# Base de Conhecimento de Linguagens de Programação

Este projeto é uma simples base de conhecimento de linguagens de programação, onde o usuário pode pesquisar por uma linguagem e obter informações sobre ela.

## Funcionalidades

- Pesquisar por linguagens de programação.
- Visualizar informações como nome, descrição, ano de criação e um link para a documentação.

## Tecnologias Utilizadas

- HTML
- CSS
- JavaScript
