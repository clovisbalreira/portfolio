# Base de Conhecimento
📌 **Descrição**
Uma aplicação de pesquisa que serve como base de conhecimento sobre diversas linguagens de programação.

🚀 **Tecnologias**
- HTML
- CSS
- JavaScript

🎯 **Funcionalidades**
- Pesquisar linguagens de programação por nome.
- Visualização de informações detalhadas (nome, descrição, ano de criação).
- Links diretos para documentações oficiais.

🖥️ **Demonstração**
Link do projeto online: https://clovisbalreira.github.io/portfolio/websites/alura/base-de-conhecimento/index.html
