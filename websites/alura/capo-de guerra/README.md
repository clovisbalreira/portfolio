# Capo de Guerra
📌 **Descrição**
Jogo interativo onde você monta seu time de heróis para enfrentar um time de vilões com forças aleatórias.

🚀 **Tecnologias**
- HTML
- CSS
- JavaScript

🎯 **Funcionalidades**
- Criação dinâmica de time com até 10 personagens.
- Geração aleatória de força para cada herói e vilão.
- Sistema de batalha com soma de forças e determinação de vencedor.
- Funcionalidade para reiniciar o jogo.

🖥️ **Demonstração**
