# Chão de Vidro
📌 **Descrição**
Jogo interativo de estratégia e coragem onde o jogador deve atravessar um caminho de pisos de vidro escolhendo corretamente os não quebráveis.

🚀 **Tecnologias**
- HTML
- CSS
- JavaScript

🎯 **Funcionalidades**
- Escolha de nome e número de rodadas.
- Geração dinâmica do caminho de pisos coloridos.
- Sistema de progresso por rodadas.
- Mensagens de feedback e reset de jogo ao perder ou vencer.

🖥️ **Demonstração**
