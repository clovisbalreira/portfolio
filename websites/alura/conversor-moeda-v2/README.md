# Conversor de Moeda V2
📌 **Descrição**
Uma versão divertida de conversor de moedas com temática espacial e visual interativo.

🚀 **Tecnologias**
- HTML
- CSS
- JavaScript

🎯 **Funcionalidades**
- Conversão de Reais para Dólar e Euro.
- Interação temática relacionada à velocidade da luz e espaço sideral.
- Design visual diferenciado e imersivo.

🖥️ **Demonstração**
Link do projeto online: https://clovisbalreira.github.io/portfolio/websites/alura/conversor-moeda-v2/index.html
