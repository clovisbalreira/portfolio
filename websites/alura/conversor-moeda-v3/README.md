# Conversor de Moeda V3
📌 **Descrição**
Conversor focado na moeda sul-coreana (Won), permitindo calcular seu equivalente em Reais com taxas personalizáveis.

🚀 **Tecnologias**
- HTML
- CSS
- JavaScript

🎯 **Funcionalidades**
- Entrada de valor em Wons.
- Definição manual de taxa de cotação.
- Cálculo de equivalência em Reais com ajuste de casas decimais.

🖥️ **Demonstração**
