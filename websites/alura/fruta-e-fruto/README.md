# Fruta e Fruto
📌 **Descrição**
Site de gastronomia desenvolvido utilizando o framework Bootstrap para garantir responsividade e design moderno.

🚀 **Tecnologias**
- HTML
- CSS
- JavaScript
- Bootstrap

🎯 **Funcionalidades**
- Página inicial com informações sobre pratos e receitas.
- Galeria de fotos responsiva.
- Formulário de contato estruturado.
