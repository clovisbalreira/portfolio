# Gatito
📌 **Descrição**
Site temático sobre gatos que utiliza efeitos visuais como parallax e carrossel para uma navegação moderna.

🚀 **Tecnologias**
- HTML
- CSS
- JavaScript

🎯 **Funcionalidades**
- Página inicial com informações curiosas sobre gatos.
- Galeria de fotos com visualização em carrossel.
- Formulário de contato estilizado.

🖥️ **Demonstração**
Link do projeto online: https://clovisbalreira.github.io/portfolio/websites/alura/gatito/index.html
