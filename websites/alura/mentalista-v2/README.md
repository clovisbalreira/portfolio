# Mentalista V2
📌 **Descrição**
Uma versão avançada do Mentalista com maior intervalo de números e limite de tentativas para aumentar o desafio.

🚀 **Tecnologias**
- HTML
- CSS
- JavaScript

🎯 **Funcionalidades**
- Intervalo de adivinhação ampliado (1 a 1000).
- Configuração personalizada da quantidade de tentativas.
- Histórico visual de chutes realizados para guiar o jogador.
- Dicas comparativas para cada palpite inserido.

🖥️ **Demonstração**
Link do projeto online: https://clovisbalreira.github.io/portfolio/websites/alura/mentalista-v2/index.html
