# Seletor de Perfis Netflix
📌 **Descrição**
Aplicação que simula a interface de seleção de perfis da Netflix, com recursos de personalização visual dinâmica.

🚀 **Tecnologias**
- HTML
- CSS
- JavaScript

🎯 **Funcionalidades**
- Seleção de perfis por categorias (Filmes, Músicas, Variado).
- Troca dinâmica de temas (Modo Claro/Escuro e temas Netflix/Spotify).
- Carregamento dinâmico de cartões de perfil via JavaScript.
- Design totalmente responsivo.

🖥️ **Demonstração**
Link do projeto online: https://clovisbalreira.github.io/portfolio/websites/alura/netflix/index.html
