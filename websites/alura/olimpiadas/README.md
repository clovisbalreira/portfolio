# Olimpíadas
📌 **Descrição**
Sua central de informações sobre atletas olímpicos, permitindo encontrar detalhes sobre nome, modalidade e conquistas.

🚀 **Tecnologias**
- HTML
- CSS
- JavaScript

🎯 **Funcionalidades**
- Busca dinâmica de atletas por nome ou modalidade.
- Listagem dos atletas mais compatíveis com a pesquisa realizada.
- Link direto para a página oficial de cada atleta com mais informações.

🖥️ **Demonstração**
