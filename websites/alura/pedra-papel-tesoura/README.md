# Pedra, Papel ou Tesoura
📌 **Descrição**
Jogo clássico de "Pedra, Papel e Tesoura" com uma versão estendida que inclui "Lagarto e Spock", inspirado na cultura pop.

🚀 **Tecnologias**
- HTML
- CSS
- JavaScript

🎯 **Funcionalidades**
- Verificação inicial de idade para o jogador.
- Modos de jogo variados com múltiplas escolhas de disputa.
- Lógica de jogo contra o computador com escolha aleatória de jogada.
- Interface animada e feedback visual do resultado.

🖥️ **Demonstração**
Link do projeto online: https://clovisbalreira.github.io/portfolio/websites/alura/pedra-papel-tesoura/index.html
