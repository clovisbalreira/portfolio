# Portfólio Imersão Dev
📌 **Descrição**
Meu portfólio pessoal reunindo os projetos desenvolvidos durante a Imersão Dev da Alura de forma visual e organizada.

🚀 **Tecnologias**
- HTML
- CSS
- JavaScript

🎯 **Funcionalidades**
- Listagem completa de projetos e desafios com links de acesso.
- Sistema de alternância entre temas Claro e Escuro.
- Visualização de vídeos demonstrativos para cada projeto.
- Layout responsivo e design moderno.
