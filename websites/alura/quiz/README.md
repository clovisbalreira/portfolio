# Quiz Interativo
📌 **Descrição**
Jogo de perguntas e respostas dinâmico que desafia seus conhecimentos com diversos sistemas de ajuda.

🚀 **Tecnologias**
- HTML
- CSS
- JavaScript

🎯 **Funcionalidades**
- 10 perguntas e alternativas sorteadas e embaralhadas aleatoriamente por partida.
- Sistemas de ajuda: Pular Questão, Ajuda dos Universitários e Cartas (Naipe).
- Tela final com resumo de desempenho e feedback visual das respostas.
- Tabela de acertos e erros detalhada.

🖥️ **Demonstração**
Link do projeto online: https://clovisbalreira.github.io/portfolio/websites/alura/quiz/index.html
