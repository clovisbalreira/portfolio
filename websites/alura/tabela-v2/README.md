# Tabela de Classificação V2
📌 **Descrição**
Versão simplificada e divertida para registro rápido de pontuações e resultados entre jogadores.

🚀 **Tecnologias**
- HTML
- CSS
- JavaScript

🎯 **Funcionalidades**
- Cadastro rápido de novos jogadores.
- Botões de ação para Vitórias, Empates e Derrotas com atualização instantânea da tabela.
- Atribuição automática de emojis aleatórios para cada participante.
- Funcionalidade para zerar o placar e reiniciar a disputa.

🖥️ **Demonstração**
