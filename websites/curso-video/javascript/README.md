# JavaScript Exercises - Curso em Vídeo
📌 **Descrição**
Coleção de exercícios e projetos práticos desenvolvidos durante o curso de JavaScript do canal Curso em Vídeo, ministrado pelo professor Gustavo Guanabara.

🚀 **Tecnologias**
- HTML
- CSS
- JavaScript

🎯 **Funcionalidades**
- Exercícios de lógica de programação e algoritmos
- Manipulação avançada do DOM (Document Object Model)
- Uso de variáveis compostas (Arrays) e Objetos
- Implementação de funções, parâmetros e recursividade
- Projetos interativos com eventos de interface

🖥️ **Demonstração**
Link do projeto online: https://clovisbalreira.github.io/portfolio/websites/curso-video/javascript/index.html
