# Meu Primeiro Programa
📌 **Descrição**
Primeiro contato com JavaScript, usando alert, confirm e prompt para interação básica com o usuário.

🚀 **Tecnologias**
- HTML
- CSS
- JavaScript

🎯 **Funcionalidades**
- Exibição de alertas (window.alert)
- Confirmação de ação (window.confirm)
- Entrada de dados via prompt (window.prompt)
