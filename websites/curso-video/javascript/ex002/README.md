# Variáveis
📌 **Descrição**
Estudo sobre os tipos de variáveis e dados em JavaScript, utilizando a função typeof para identificação.

🚀 **Tecnologias**
- HTML
- CSS
- JavaScript

🎯 **Funcionalidades**
- Identificação do tipo de dado (typeof)
- Entrada de dados via prompt
- Manipulação do DOM para exibir o tipo da variável
