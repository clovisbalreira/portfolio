# Soma de Variáveis
📌 **Descrição**
Exercício de conversão de tipos de dados (string para number) e realização de operações matemáticas simples.

🚀 **Tecnologias**
- HTML
- CSS
- JavaScript

🎯 **Funcionalidades**
- Conversão de string para float (Number.parseFloat)
- Soma de dois valores numéricos
- Exibição do resultado no corpo da página

🖥️ **Demonstração**
Link do projeto online: https://clovisbalreira.github.io/portfolio/websites/curso-video/javascript/ex003/index.html

▶️ **Como rodar**
