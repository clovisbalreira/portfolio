# Formatando String
📌 **Descrição**
Manipulação de strings para exibição formatada, explorando propriedades e métodos nativos do JavaScript.

🚀 **Tecnologias**
- HTML
- CSS
- JavaScript

🎯 **Funcionalidades**
- Contagem de caracteres (length)
- Conversão para maiúsculas (toUpperCase)
- Conversão para minúsculas (toLowerCase)
