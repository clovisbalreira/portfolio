# Formatando Números
📌 **Descrição**
Formatação de valores numéricos para diferentes moedas e padrões decimais.

🚀 **Tecnologias**
- HTML
- CSS
- JavaScript

🎯 **Funcionalidades**
- Limitação de casas decimais (toFixed)
- Formatação de moeda (toLocaleString) para BRL, USD e EUR
