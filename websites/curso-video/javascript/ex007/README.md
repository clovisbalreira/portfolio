# Primeiros passos com o DOM
📌 **Descrição**
Introdução ao Document Object Model (DOM) e como interagir com elementos HTML através do JavaScript.

🚀 **Tecnologias**
- HTML
- CSS
- JavaScript

🎯 **Funcionalidades**
- Seleção de elementos via querySelector e getElementByTagName
- Alteração de estilos dinamicamente
- Leitura de propriedades globais (window.navigator, window.document.URL)

🖥️ **Demonstração**
