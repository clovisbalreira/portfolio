# Eventos DOM
📌 **Descrição**
Manipulação de eventos de mouse diretamente em elementos HTML para interatividade.

🚀 **Tecnologias**
- HTML
- CSS
- JavaScript

🎯 **Funcionalidades**
- Reação ao clique (click)
- Interação ao passar o mouse (mouseenter)
- Alteração visual ao retirar o mouse (mouseout)
