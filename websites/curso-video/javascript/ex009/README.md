# Somando Valores
📌 **Descrição**
Aplicação simples que permite ao usuário somar dois valores e visualizar o resultado em tempo real.

🚀 **Tecnologias**
- HTML
- CSS
- JavaScript

🎯 **Funcionalidades**
- Coleta de dados via inputs de formulário
- Cálculo matemático simples no lado do cliente
- Exibição dinâmica do resultado em uma div específica

🖥️ **Demonstração**
Link do projeto online: https://clovisbalreira.github.io/portfolio/websites/curso-video/javascript/ex009/index.html
