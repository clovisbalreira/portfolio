# Condições
📌 **Descrição**
Exercícios práticos envolvendo estruturas condicionais simples, compostas, alinhadas e de múltipla escolha.

🚀 **Tecnologias**
- HTML
- CSS
- JavaScript

🎯 **Funcionalidades**
- Verificação de velocidade de carro
- Identificação de nacionalidade
- Verificador de obrigatoriedade de voto
- Saudação baseada no horário atual e dia da semana

🖥️ **Demonstração**
