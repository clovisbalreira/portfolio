# Hora do Dia
📌 **Descrição**
Página dinâmica que detecta a hora atual e altera a imagem e a cor de fundo automaticamente.

🚀 **Tecnologias**
- HTML
- CSS
- JavaScript

🎯 **Funcionalidades**
- Detecta a hora atual do sistema
- Altera a imagem exibida (Manhã, Tarde, Noite)
- Altera a cor de fundo dinamicamente via JavaScript

🖥️ **Demonstração**
Link do projeto online: https://clovisbalreira.github.io/portfolio/websites/curso-video/javascript/ex011/index.html
