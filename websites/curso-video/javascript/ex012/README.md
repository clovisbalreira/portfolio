# Verificador de idade
📌 **Descrição**
Ferramenta que calcula a idade de uma pessoa e exibe uma imagem baseada na idade e no sexo informado.

🚀 **Tecnologias**
- HTML
- CSS
- JavaScript

🎯 **Funcionalidades**
- Cálculo automático de idade a partir do ano informado
- Verificação de gênero via inputs radio
- Exibição de imagem correspondente ao perfil detectado

🖥️ **Demonstração**
