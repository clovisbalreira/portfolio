# Repetições
📌 **Descrição**
Estudo sobre as diversas formas de implementar laços de repetição em JavaScript.

🚀 **Tecnologias**
- HTML
- CSS
- JavaScript

🎯 **Funcionalidades**
- Exemplificação do laço While (controle no início)
- Exemplificação do laço Do While (controle no final)
- Exemplificação do laço For (com variável de controle)
