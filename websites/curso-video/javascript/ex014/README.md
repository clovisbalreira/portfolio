# Vamos contar
📌 **Descrição**
Aplicativo de contagem personalizável que permite definir início, fim e o passo (incremento/decremento).

🚀 **Tecnologias**
- HTML
- CSS
- JavaScript

🎯 **Funcionalidades**
- Contagem crescente ou decrescente
- Validação automática do valor de 'passo'
- Exibição do resultado dinamicamente com uso de emojis

🖥️ **Demonstração**
Link do projeto online: https://clovisbalreira.github.io/portfolio/websites/curso-video/javascript/ex014/index.html
