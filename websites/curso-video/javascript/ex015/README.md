# Tabuada
📌 **Descrição**
Gerador de tabuada rápido e eficiente, permitindo ao usuário escolher qualquer número.

🚀 **Tecnologias**
- HTML
- CSS
- JavaScript

🎯 **Funcionalidades**
- Cálculo automático da tabuada do 1 ao 10
- Exibição dos resultados dentro de um componente select
- Limpeza automática de cálculos anteriores ao gerar novos

🖥️ **Demonstração**
Link do projeto online: https://clovisbalreira.github.io/portfolio/websites/curso-video/javascript/ex015/index.html
