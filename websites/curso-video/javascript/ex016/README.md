# Variáveis Compostas
📌 **Descrição**
Exercício focado no aprendizado e manipulação de arrays (variáveis compostas) em JavaScript.

🚀 **Tecnologias**
- HTML
- CSS
- JavaScript

🎯 **Funcionalidades**
- Adição de valores em posições específicas e ao final do array
- Verificação do tamanho do array
- Ordenação de valores
- Iteração sobre arrays usando `for` e `for...in`
- Busca de índices por valor

🖥️ **Demonstração**
Link do projeto online: https://clovisbalreira.github.io/portfolio/websites/curso-video/javascript/ex016/index.html
