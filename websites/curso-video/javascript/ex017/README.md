# Funções em JavaScript
📌 **Descrição**
Exercício para praticar a criação e uso de funções, cobrindo desde lógica básica até recursividade.

🚀 **Tecnologias**
- HTML
- CSS
- JavaScript

🎯 **Funcionalidades**
- Verificação de paridade (par ou ímpar)
- Soma de valores com parâmetros opcionais
- Utilização de funções anônimas e arrow functions
- Cálculo de fatorial de forma iterativa e recursiva

🖥️ **Demonstração**
