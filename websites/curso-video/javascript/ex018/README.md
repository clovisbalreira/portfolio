# Analisador de Números
📌 **Descrição**
Uma aplicação interativa que coleta números inseridos pelo usuário, valida as entradas e fornece estatísticas detalhadas sobre o conjunto de dados.

🚀 **Tecnologias**
- HTML
- CSS
- JavaScript

🎯 **Funcionalidades**
- Adição de números entre 1 e 100 com validação de duplicatas
- Listagem dinâmica dos valores adicionados
- Cálculo da soma total e média dos valores
- Identificação do maior e menor valor informado
- Verificação da quantidade de números cadastrados
