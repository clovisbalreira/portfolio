# Objetos em JavaScript
📌 **Descrição**
Exercício introdutório para entender a estrutura de objetos, propriedades e métodos em JavaScript.

🚀 **Tecnologias**
- HTML
- CSS
- JavaScript

🎯 **Funcionalidades**
- Criação de objetos com múltiplas propriedades
- Implementação de métodos internos (ex: função para alterar peso)
- Atualização dinâmica de informações no HTML com base no estado do objeto

🖥️ **Demonstração**
Link do projeto online: https://clovisbalreira.github.io/portfolio/websites/curso-video/javascript/ex019/index.html
