# Lâmpada Interativa
📌 **Descrição**
Um projeto visual interativo que simula o comportamento de uma lâmpada que reage às ações do mouse.

🚀 **Tecnologias**
- HTML
- CSS
- JavaScript

🎯 **Funcionalidades**
- Acender a lâmpada ao passar o mouse por cima
- Apagar a lâmpada ao retirar o mouse
- Quebrar a lâmpada permanentemente ao clicar
- Exibição dinâmica da data e hora atual

🖥️ **Demonstração**
Link do projeto online: https://clovisbalreira.github.io/portfolio/websites/curso-video/javascript/lambada/index.html
