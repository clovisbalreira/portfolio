# Dev em Dobro - Coleção de Projetos
📌 **Descrição**
Esta é uma coleção de projetos desenvolvidos durante os eventos e cursos do canal "Dev em Dobro". A coleção abrange desde landing pages promocionais e galerias de personagens até interfaces interativas complexas, focando no aprendizado prático de tecnologias frontend.

🚀 **Tecnologias**
- HTML
- CSS
- JavaScript

🎯 **Funcionalidades**
- Diversas implementações de carrosséis e sliders.
- Galerias de personagens interativas (X-Men, Dragon Ball, One Piece).
- Landing pages responsivas (GTA V, Super Mario, Horror Game).
- Interfaces com troca de temas e filtros dinâmicos.

🖥️ **Demonstração**
Link da coleção online: https://seu-link-aqui.com

📸 **Imagens**

▶️ **Como rodar**
1. Clone o repositório.
2. Navegue até a pasta do projeto desejado.
3. Abra o arquivo `index.html` no seu navegador.

📚 **Aprendizados**
Esta coleção representa uma jornada de aprendizado intensiva em desenvolvimento frontend, onde foram explorados conceitos fundamentais e avançados de:
- Estruturação semântica com HTML5.
- Estilização moderna e responsiva com CSS3 (Flexbox, Grid, Variáveis).
- Interatividade e manipulação do DOM com JavaScript Vanilla.
- Experiência do usuário (UX) e design de interfaces (UI).
