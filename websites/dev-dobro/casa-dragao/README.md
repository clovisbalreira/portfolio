# Casa do Dragão
📌 **Descrição**
Um projeto que apresenta um carrossel interativo com os dragões da série "A Casa do Dragão" (House of the Dragon), exibindo informações detalhadas sobre cada criatura.

🚀 **Tecnologias**
- HTML
- CSS
- JavaScript

🎯 **Funcionalidades**
- Carrossel de imagens dinâmico.
- Transição suave entre as informações dos dragões.
- Seleção manual através de botões de navegação.

🖥️ **Demonstração**
Link do projeto online: https://clovisbalreira.github.io/portfolio/websites/dev-dobro/casa-dragao/index.html
