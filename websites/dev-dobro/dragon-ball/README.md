# Dragon Ball Gallery
📌 **Descrição**
Uma galeria interativa que exibe personagens do universo Dragon Ball, permitindo aos usuários explorar diferentes guerreiros.

🚀 **Tecnologias**
- HTML
- CSS
- JavaScript

🎯 **Funcionalidades**
- Listagem dinâmica de personagens.
- Interface responsiva adaptada para diferentes dispositivos.
- Manipulação do DOM para exibição de conteúdo.

🖥️ **Demonstração**
Link do projeto online: https://clovisbalreira.github.io/portfolio/websites/dev-dobro/dragon-ball/index.html
