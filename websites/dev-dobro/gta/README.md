# GTA V Landing Page
📌 **Descrição**
Uma página de destino (landing page) inspirada no jogo Grand Theft Auto V, apresentando informações sobre o jogo e opções de compra para diversas plataformas.

🚀 **Tecnologias**
- HTML
- CSS
- JavaScript

🎯 **Funcionalidades**
- Menu de navegação fixo e responsivo.
- Botão interativo para seleção de plataformas.
- Design visual fiel à identidade da Rockstar Games.

🖥️ **Demonstração**
Link do projeto online: https://clovisbalreira.github.io/portfolio/websites/dev-dobro/gta/index.html
