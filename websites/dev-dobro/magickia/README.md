# Magick IA
📌 **Descrição**
Uma loja virtual de cartas Magic, geradas por inteligência artificial, que permite filtrar produtos por categoria e preço.

🚀 **Tecnologias**
- HTML
- CSS
- JavaScript

🎯 **Funcionalidades**
- Filtro por categorias de raridade (Comum, Rara, Épica).
- Filtro por preço máximo dinâmico.
- Renderização dinâmica de cartas a partir de arquivos JavaScript.

🖥️ **Demonstração**
Link do projeto online: https://clovisbalreira.github.io/portfolio/websites/dev-dobro/magickia/index.html
