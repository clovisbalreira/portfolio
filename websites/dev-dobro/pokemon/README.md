# Pokémon Listagem
📌 **Descrição**
Uma listagem de Pokémons da primeira geração com suporte a troca de temas (claro e escuro).

🚀 **Tecnologias**
- HTML
- CSS
- JavaScript

🎯 **Funcionalidades**
- Listagem de Pokémons com GIFs e informações básicas.
- Alteração de tema (Light/Dark mode) com persistência visual.
- Scrollbar personalizada e estilizada.

🖥️ **Demonstração**
Link do projeto online: https://clovisbalreira.github.io/portfolio/websites/dev-dobro/pokemon/index.html
