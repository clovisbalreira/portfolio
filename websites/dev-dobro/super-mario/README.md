# Super Mario Bros - O Filme
📌 **Descrição**
Uma landing page promocional do filme Super Mario Bros, com vídeo de fundo e trailer interativo.

🚀 **Tecnologias**
- HTML
- CSS
- JavaScript

🎯 **Funcionalidades**
- Vídeo de fundo em loop automático.
- Modal interativo para visualização do trailer.
- Layout responsivo e visualmente impactante.

🖥️ **Demonstração**
Link do projeto online: https://clovisbalreira.github.io/portfolio/websites/dev-dobro/super-mario/index.html
