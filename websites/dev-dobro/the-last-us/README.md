# The Last of Us - Série
📌 **Descrição**
Um projeto dedicado à série The Last of Us da HBO, apresentando um carrossel de imagens com cenas marcantes.

🚀 **Tecnologias**
- HTML
- CSS
- JavaScript

🎯 **Funcionalidades**
- Carrossel de imagens com transições suaves.
- Navegação por botões inferiores.
- Menu lateral com links para redes sociais e GitHub.

🖥️ **Demonstração**
Link do projeto online: https://clovisbalreira.github.io/portfolio/websites/dev-dobro/the-last-us/index.html
