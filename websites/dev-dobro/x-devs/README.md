# X-DEVS Slider
📌 **Descrição**
Um slider interativo apresentando os "X-DEVS", desenvolvedores com habilidades inspiradas em mutantes.

🚀 **Tecnologias**
- HTML
- CSS
- JavaScript

🎯 **Funcionalidades**
- Navegação entre cartões de personagens com setas (Avançar/Voltar).
- Troca de fundo e conteúdo dinamicamente.
- Design responsivo e temático.

🖥️ **Demonstração**
Link do projeto online: https://clovisbalreira.github.io/portfolio/websites/dev-dobro/x-devs/index.html
