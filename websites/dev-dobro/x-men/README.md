# X-Men Character Selection
📌 **Descrição**
Uma interface de seleção de personagens inspirada no universo X-Men, permitindo visualizar detalhes de cada mutante.

🚀 **Tecnologias**
- HTML
- CSS
- JavaScript

🎯 **Funcionalidades**
- Seleção interativa de personagens ao passar o mouse.
- Atualização dinâmica da imagem principal, nome e descrição.
- Grid de cards responsivo.

🖥️ **Demonstração**
Link do projeto online: https://clovisbalreira.github.io/portfolio/websites/dev-dobro/x-men/index.html
