# Jogo Ping Pong
📌 **Descrição**
O clássico jogo Ping Pong construído inteiramente com JavaScript e HTML5 Canvas, proporcionando uma experiência interativa e desafiadora contra o computador.

🚀 **Tecnologias**
- HTML
- CSS
- JavaScript

🎯 **Funcionalidades**
- Controle da raquete pelo mouse em tempo real
- Adversário automático (IA básica) com velocidade adaptável
- Sistema de pontuação dinâmico e aumento progressivo de dificuldade
