# Netflix Clone
📌 **Descrição**
Recriação da interface principal da Netflix desenvolvida durante o bootcamp da Digital Innovation One (DIO). O projeto foca no design responsivo e no uso de carrosséis para exibição de filmes.

🚀 **Tecnologias**
- HTML
- CSS
- JavaScript

🎯 **Funcionalidades**
- Interface inspirada na Netflix com destaque para filme principal.
- Carrossel de filmes interativo utilizando Owl Carousel.
- Design totalmente responsivo para diferentes dispositivos.

🖥️ **Demonstração**
Link do projeto online: https://clovisbalreira.github.io/portfolio/websites/dio/netflix/index.html

Neste projeto, aprendi a utilizar a biblioteca Owl Carousel para criar sliders interativos, aprimorei o uso de Flexbox para layouts complexos e explorei técnicas de design responsivo com Media Queries.
