# Círculo de Progresso
📌 **Descrição**
Uma aplicação interativa que exibe um gráfico circular de progresso. O usuário insere um valor numérico e o círculo se preenche proporcionalmente ao valor informado.

🚀 **Tecnologias**
- HTML
- CSS
- JavaScript

🎯 **Funcionalidades**
- Entrada de valor numérico (0 a 100).
- Animação visual do preenchimento do círculo via SVG.
- Exibição dinâmica do valor percentual.

🖥️ **Demonstração**
