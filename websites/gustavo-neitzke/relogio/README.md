# Relógios e Cronômetro
📌 **Descrição**
Uma interface que combina três tipos de ferramentas de medição de tempo: um relógio digital, um relógio analógico e um cronômetro completo com marcação de voltas.

🚀 **Tecnologias**
- HTML
- CSS
- JavaScript

🎯 **Funcionalidades**
- Relógio digital com atualização em tempo real.
- Relógio analógico com ponteiros funcionais.
- Cronômetro com funções de iniciar, pausar, reiniciar e marcar voltas.

🖥️ **Demonstração**
Link do projeto online: https://clovisbalreira.github.io/portfolio/websites/gustavo-neitzke/relogio/index.html
