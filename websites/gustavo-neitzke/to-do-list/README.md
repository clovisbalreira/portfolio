# To Do List
📌 **Descrição**
Uma lista de tarefas clássica que permite ao usuário adicionar, marcar como concluídas e excluir tarefas, ajudando na organização pessoal.

🚀 **Tecnologias**
- HTML
- CSS
- JavaScript

🎯 **Funcionalidades**
- Adição de novas tarefas via campo de texto.
- Listagem dinâmica das tarefas criadas.
- Validação de entrada para evitar tarefas vazias.

🖥️ **Demonstração**
Link do projeto online: https://clovisbalreira.github.io/portfolio/websites/gustavo-neitzke/to-do-list/index.html
