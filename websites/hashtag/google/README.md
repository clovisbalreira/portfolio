# Google

📌 Descrição
Clone da página inicial do buscador Google, focado em replicar a interface visual e os elementos de navegação.

🚀 Tecnologias
- HTML
- CSS

🎯 Funcionalidades
- Interface fiel ao Google.
- Barra de pesquisa centralizada.
- Menu de navegação superior (Gmail, Imagens).
- Botões de ação (Pesquisa Google e Sinto-me com sorte).
- Rodapé com links institucionais.
