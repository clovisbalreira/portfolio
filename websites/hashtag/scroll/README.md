# Scroll

📌 Descrição
Projeto focado em animação de scroll horizontal infinito, ideal para exibição de logos, habilidades ou parceiros.

🚀 Tecnologias
- HTML
- CSS
- JavaScript

🎯 Funcionalidades
- Scroll horizontal contínuo e infinito.
- Exibição de lista de habilidades (Excel, Python, SQL, etc.).
- Animação suave controlada por CSS/JavaScript.
