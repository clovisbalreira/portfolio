# Calculadora Básica

📌 Descrição
Uma calculadora simples desenvolvida para realizar operações matemáticas básicas entre dois números.

🚀 Tecnologias
- HTML
- CSS
- JavaScript

🎯 Funcionalidades
- Entrada de dois valores numéricos.
- Cálculo automático do resultado.
- Interface simples e intuitiva.
