# Calculadora Desconto

📌 Descrição
Ferramenta prática para calcular rapidamente o valor de desconto de um produto e o preço final.

🚀 Tecnologias
- HTML
- CSS
- JavaScript

🎯 Funcionalidades
- Inserir o valor original do produto.
- Inserir a porcentagem de desconto desejada.
- Cálculo instantâneo do desconto aplicado.
