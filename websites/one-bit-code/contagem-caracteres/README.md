# Contagem de Caracteres

📌 Descrição
Um utilitário simples que conta a quantidade de caracteres de um texto inserido pelo usuário em tempo real.

🚀 Tecnologias
- HTML
- CSS
- JavaScript

🎯 Funcionalidades
- Campo de texto para inserção de frases.
- Contador automático de caracteres.
- Feedback imediato ao usuário.
