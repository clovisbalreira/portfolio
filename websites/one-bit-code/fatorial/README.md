# Fatorial

📌 Descrição
Calculadora de fatorial desenvolvida para exercitar a lógica de programação e recursividade/laços de repetição.

🚀 Tecnologias
- HTML
- CSS
- JavaScript

🎯 Funcionalidades
- Cálculo de fatorial de números inteiros.
- Interface limpa para exibição do resultado.
