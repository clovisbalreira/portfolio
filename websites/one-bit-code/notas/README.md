# Notas

📌 Descrição
Aplicação para gerenciamento de notas escolares e cálculo automático da média.

🚀 Tecnologias
- HTML
- CSS
- JavaScript

🎯 Funcionalidades
- Adicionar múltiplas notas dinamicamente.
- Cálculo da média aritmética simples.
- Exibição de resultados em tempo real.
