# Petlive

📌 Descrição
Landing page moderna para a clínica veterinária PetLife, focada em conversão e apresentação de serviços.

🚀 Tecnologias
- HTML
- CSS
- JavaScript

🎯 Funcionalidades
- Seções de Início, Clínica, Farmácia e Dúvidas.
- Menu de navegação fixo.
- Seção de FAQ (Perguntas Frequentes) interativa.
- Links para contato via WhatsApp e E-mail.
