# Tabuada

📌 Descrição
Gerador automático de tabuada de multiplicação para facilitar o aprendizado e consulta rápida.

🚀 Tecnologias
- HTML
- CSS
- JavaScript

🎯 Funcionalidades
- Geração completa da tabuada de 1 a 10 para o número escolhido.
- Exibição limpa dos cálculos na tela.
