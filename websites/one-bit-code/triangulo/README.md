# Triangulo

📌 Descrição
Ferramenta para cálculo e verificação de propriedades de triângulos com base no comprimento de seus lados.

🚀 Tecnologias
- HTML
- CSS
- JavaScript

🎯 Funcionalidades
- Entrada para os três lados do triângulo.
- Verificação da validade do triângulo.
- Processamento de cálculos geométricos.
