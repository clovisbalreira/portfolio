# Apresentação

📌 Descrição
Uma página de apresentação pessoal interativa que centraliza links importantes, currículo e galeria de imagens.

🚀 Tecnologias
- HTML
- CSS
- JavaScript

🎯 Funcionalidades
- Navegação intuitiva entre seções (Links, Currículo, Imagens)
- Interface interativa com elementos clicáveis
- Acesso rápido a formulário de comentários externo

🖥️ Demonstração
Link do projeto online: https://clovisbalreira.github.io/portfolio/websites/pessoal/apresentacao/index.html
