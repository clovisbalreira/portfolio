# 📄 Gerador de Currículo Multilíngue

## 📌 Sobre o projeto
Aplicação web dinâmica que gera um currículo interativo com suporte a múltiplos idiomas (português, inglês e espanhol), permitindo visualização e filtragem de cursos e experiências.

## 🚀 Tecnologias utilizadas
- HTML5
- CSS3
- JavaScript

## 🎯 Funcionalidades
- Alternância entre idiomas (PT, EN, ES)
- Geração dinâmica de currículo via JavaScript
- Filtros por cursos, ano, instituição e tipo
- Exibição interativa de dados profissionais
- Interface responsiva

## ▶️ Como usar

- Selecione o idioma desejado
- Navegue pelos filtros
- Visualize cursos e experiências dinamicamente

## 🗂️ Estrutura dos dados

Os dados são organizados em objetos JavaScript contendo:
- Nome do curso
- Instituição
- Ano
- Tipo
- Traduções (PT, EN, ES)

## ⚙️ Destaques técnicos
- Manipulação avançada do DOM
- Estrutura modular com múltiplas funções reutilizáveis
- Organização de dados complexos em objetos
- Sistema de filtragem dinâmica
