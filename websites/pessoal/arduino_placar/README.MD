# Arduino Placar

📌 Descrição
Um sistema de placar eletrônico completo que integra uma interface web com hardware Arduino para monitoramento de jogos em tempo real.

🚀 Tecnologias
- HTML
- CSS
- JavaScript (Web Serial API)
- C++ (Arduino)

🎯 Funcionalidades
- Comunicação em tempo real com hardware Arduino via Serial
- Cronômetro configurável para tempos de jogo e intervalos
- Registro de gols com sistema de anulação (VAR) de 5 segundos
- Gestão dinâmica de múltiplas equipes e visualização de tabela de jogos

🖥️ Demonstração
Link do projeto online: https://clovisbalreira.github.io/portfolio/websites/pessoal/arduino_placar/index.html

▶️ Como rodar
1. Clone o repositório
2. Carregue o código C++ no seu Arduino
3. Abra o arquivo index.html em um navegador compatível com Web Serial

📚 Aprendizados
Implementação de comunicação Serial em navegadores, lógica complexa de cronômetro e sincronização de estado entre hardware e software.
