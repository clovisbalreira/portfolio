# Banco de Horas

📌 Descrição
Aplicativo simples e funcional para registro e cálculo de horas trabalhadas, facilitando o controle de banco de horas.

🚀 Tecnologias
- HTML
- CSS
- JavaScript

🎯 Funcionalidades
- Registro de horários de entrada e saída
- Cálculo automático de total de horas trabalhadas
- Visualização de dados em formato de tabela organizada
- Suporte para múltiplos registros diários
