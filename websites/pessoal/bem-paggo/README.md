# BemPaggo

📌 Descrição
Simulador de formulário de compra de pacotes de adesivos com geração dinâmica de nota fiscal e confirmação de pedido.

🚀 Tecnologias
- HTML
- CSS
- JavaScript

🎯 Funcionalidades
- Seleção interativa de adesivos (React, Vue, Angular)
- Controle de quantidade com botões de incremento e decremento
- Geração automática de nota fiscal com cálculo de valores unitários e totais
- Sistema de feedback para confirmação ou recusa de compra
