# Calendário Futebol Brasileiro

📌 Descrição
Visualizador interativo de calendários do futebol brasileiro, permitindo comparar o cenário atual com propostas alternativas e futuras.

🚀 Tecnologias
- HTML
- CSS
- JavaScript (ES6 Modules)

🎯 Funcionalidades
- Alternância entre 4 cenários de calendário (Atual, Modificado, 2026, Europeu)
- Visualização dinâmica de campeonatos e períodos de disputa
- Interface interativa com rádio buttons para troca de contexto
- Legendas detalhadas para identificação de competições
