# CDD - Contabilidade Doméstica

📌 Descrição
Sistema de contabilidade doméstica projetado para auxiliar no controle de despesas mensais e organização financeira por ano.

🚀 Tecnologias
- HTML
- CSS
- JavaScript (ES6 Modules)

🎯 Funcionalidades
- Filtragem de despesas por mês e ano através de menus suspensos
- Visualização de gastos organizada em cartões interativos
- Identificação clara de pagadores e categorias de custos
- Interface limpa e focada na usabilidade financeira
