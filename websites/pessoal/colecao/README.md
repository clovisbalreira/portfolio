# Coleção Moedas Antigas

📌 Descrição
Um catálogo digital interativo para colecionadores de moedas e cédulas antigas, com suporte a múltiplas moedas e países.

🚀 Tecnologias
- HTML
- CSS
- JavaScript

🎯 Funcionalidades
- Filtragem avançada por País, Ano, Tipo de Moeda e Valor
- Integração com APIs para busca de informações e cotações
- Visualização detalhada de itens da coleção com informações históricas
- Sistema de busca por itens únicos e categorias específicas

🖥️ Demonstração
