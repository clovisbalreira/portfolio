# Dev Midia - Travel

📌 Descrição
Landing page moderna de uma agência de viagens, focada em busca de destinos, ofertas e experiência do usuário.

🚀 Tecnologias
- HTML
- CSS
- JavaScript

🎯 Funcionalidades
- Sistema de busca de viagens (Flight, Hotel, Rental) com filtros de origem e destino
- Seção de ofertas em destaque com cálculos de desconto visual
- Galeria interativa de destinos populares ao redor do mundo
- Formulários de Newsletter e Contato integrados para conversão
