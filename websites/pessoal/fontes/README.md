# Fontes

📌 Descrição
Uma ferramenta útil para desenvolvedores e designers visualizarem como uma palavra ou frase se comporta em diferentes fontes do sistema.

🚀 Tecnologias
- HTML
- CSS
- JavaScript

🎯 Funcionalidades
- Pré-visualização instantânea em mais de 40 fontes do sistema
- Personalização de tamanho da fonte e cor via inputs interativos
- Tabela comparativa que exibe a palavra de teste ao lado do nome da fonte aplicada
- Renderização dinâmica baseada em iteração de array
