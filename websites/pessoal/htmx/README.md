# HTMX Experience

📌 Descrição
Projeto experimental para explorar as capacidades da biblioteca HTMX no carregamento dinâmico de conteúdo e interações AJAX simplificadas.

🚀 Tecnologias
- HTML
- JavaScript
- HTMX

🎯 Funcionalidades
- Carregamento de conteúdo assíncrono via requisições HTTP sem necessidade de JS complexo
- Atualização parcial da página baseada em interações do usuário (clique no botão)
- Integração simplificada de bibliotecas externas via CDN

🖥️ Demonstração
Link do projeto online: https://clovisbalreira.github.io/portfolio/websites/pessoal/htmx/index.html

▶️ Como rodar
1. Clone o repositório
2. Abra o arquivo index.html (requer um servidor local para algumas funcionalidades HTMX)

📚 Aprendizados
Entendimento do conceito de HATEOAS e como o HTMX permite criar aplicações interativas de forma declarativa diretamente no HTML.
