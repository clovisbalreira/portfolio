# Portfólio de Design (Imagens)

📌 Descrição
Galeria visual que expõe trabalhos de design gráfico, folders e projetos de identidade visual realizados.

🚀 Tecnologias
- HTML
- CSS
- JavaScript

🎯 Funcionalidades
- Exibição categorizada de projetos (Folders, Loja de CD, Realidade Aumentada)
- Alternância entre Modo Claro (Light) e Modo Escuro (Dark)
- Design responsivo com visualização otimizada para diferentes resoluções
- Integração com ícones sociais (Ionicons) e perfil do GitHub

🖥️ Demonstração
Link do projeto online: https://seu-link-aqui.com

📸 Imagens

▶️ Como rodar
1. Clone o repositório
2. Abra o arquivo index.html

📚 Aprendizados
Implementação de Dark Mode com variáveis CSS e organização de ativos de imagem em um layout de portfólio profissional.
