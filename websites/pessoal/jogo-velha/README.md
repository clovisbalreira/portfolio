# Jogo da Velha (Tic-Tac-Toe)

📌 Descrição
Um jogo da velha interativo com uma interface moderna, permitindo partidas rápidas e divertidas entre dois jogadores locais.

🚀 Tecnologias
- HTML
- CSS
- JavaScript

🎯 Funcionalidades
- Tabuleiro dinâmico com alternância automática entre os jogadores X e O
- Algoritmo de verificação de vitória para linhas, colunas e diagonais
- Detecção de empate (Velha) e feedback visual imediato do vencedor
- Botão de reinicialização rápida para novas partidas
