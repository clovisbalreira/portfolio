# Jogo da Forca

📌 Descrição
Um jogo clássico da forca com uma temática especial inspirada em Star Wars, onde cada erro aproxima o jogador do lado sombrio.

🚀 Tecnologias
- HTML
- CSS
- JavaScript

🎯 Funcionalidades
- Sistema de entrada de letras (chutes) com validação
- Representação visual do boneco (Darth Vader) que aparece conforme os erros ocorrem
- Sorteio aleatório de palavras e atualização dinâmica das lacunas
- Feedback visual de vitória ou derrota baseado nas tentativas
