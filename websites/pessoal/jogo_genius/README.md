# Jogo Genius (Simon Clone)

📌 Descrição
Um clone fiel do clássico jogo de memória Genius (ou Simon), desafiando a memória do jogador com sequências crescentes de cores e sons.

🚀 Tecnologias
- HTML
- CSS
- JavaScript

🎯 Funcionalidades
- Geração de sequências aleatórias de cores (Verde, Vermelho, Amarelo, Azul)
- Feedback visual de brilho e efeitos sonoros ao ativar cada cor
- Sistema de níveis que aumenta a dificuldade a cada acerto
- Interface circular clássica e responsiva

🖥️ Demonstração
Link do projeto online: https://clovisbalreira.github.io/portfolio/websites/pessoal/jogo_genius/index.html
