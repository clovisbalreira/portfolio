# Jogo da Memória

📌 Descrição
O clássico jogo de memória onde o objetivo é encontrar pares de cartas idênticas no menor número de tentativas possível.

🚀 Tecnologias
- HTML
- CSS
- JavaScript

🎯 Funcionalidades
- Tabuleiro de jogo gerado dinamicamente via JavaScript
- Mecânica de virada de cartas com animações suaves em CSS
- Lógica de comparação para manter cartas abertas ou fechá-las em caso de erro
- Embaralhamento aleatório das cartas a cada nova partida
