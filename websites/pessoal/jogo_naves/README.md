# Jogo Resgate (Naves)

📌 Descrição
Um jogo de ação lateral inspirado em clássicos do Atari, onde o jogador controla um helicóptero para combater inimigos e salvar aliados.

🚀 Tecnologias
- HTML
- CSS
- JavaScript (jQuery)

🎯 Funcionalidades
- Movimentação fluida do helicóptero via comandos de teclado (W, S)
- Sistema de disparos contra naves e caminhões inimigos
- Mecânica de resgate de civis com sistema de pontuação
- Efeitos sonoros e trilha sonora imersiva integrada às ações
- Detecção de colisões complexas entre múltiplos objetos
