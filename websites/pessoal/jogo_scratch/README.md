# Jogo Scratch (p5.js)

📌 Descrição
Um projeto de desenvolvimento de jogos utilizando a biblioteca p5.js, explorando gráficos 2D e interações matemáticas de colisão.

🚀 Tecnologias
- HTML
- CSS
- JavaScript (p5.js)

🎯 Funcionalidades
- Renderização de gráficos 2D via Canvas utilizando a p5.js
- Detecção de colisão entre objetos utilizando a biblioteca p5.collide2d
- Loop de jogo controlado (Draw Loop) para movimentação e física simples
- Estrutura de código organizada em sketches criativos
