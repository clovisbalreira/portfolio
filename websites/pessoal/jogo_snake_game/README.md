# Snake Game

📌 Descrição
O clássico jogo da cobrinha (Snake Game) recriado para navegadores, desafiando a agilidade e coordenação do jogador.

🚀 Tecnologias
- HTML
- CSS
- JavaScript (Canvas API)

🎯 Funcionalidades
- Movimentação fluida da cobra em quatro direções via teclado
- Sistema de pontuação que aumenta conforme a cobra consome a comida
- Geração aleatória de itens de pontuação no tabuleiro
- Detecção de colisão com as bordas do cenário e com o próprio corpo da cobra (Game Over)
