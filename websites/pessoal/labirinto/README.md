# Jogo de Labirinto

📌 Descrição
Um desafio clássico de labirinto onde o jogador deve navegar por um mapa complexo para encontrar a saída, evitando as paredes.

🚀 Tecnologias
- HTML
- JavaScript (Canvas API)

🎯 Funcionalidades
- Mapa de labirinto gerado a partir de uma matriz de dados (2D Array)
- Movimentação do jogador baseada em pixels com controle de velocidade
- Detecção de colisão precisa que impede o jogador de atravessar as paredes
- Renderização otimizada utilizando requestAnimationFrame para movimentos suaves
