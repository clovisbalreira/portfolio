# Loja Landing Page

📌 Descrição
Landing page empresarial projetada para apresentação de produtos e serviços com foco em conversão e atendimento ao cliente.

🚀 Tecnologias
- HTML
- CSS
- JavaScript

🎯 Funcionalidades
- Catálogo visual de produtos em destaque
- Formulário de contato integrado para captação de leads
- Botão flutuante de WhatsApp para atendimento direto e personalizado
- Mapa de localização interativo via iframe do Google Maps

🖥️ Demonstração
