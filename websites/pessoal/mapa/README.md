# Mapa

📌 **Descrição**
Um sistema simples de movimentação em um mapa baseado em tiles (quadrados), onde o usuário pode navegar por um cenário pré-definido.

🚀 **Tecnologias**
- HTML
- CSS
- JavaScript

🎯 **Funcionalidades**
- Movimentação do personagem utilizando as setas do teclado.
- Renderização dinâmica do cenário a partir de uma matriz de dados.
- Sistema de colisão básico que impede o movimento para áreas restritas.
- Visualização limitada ao redor do jogador (sistema de câmera/foco).

📚 **Aprendizados**
Este projeto permitiu praticar a manipulação do DOM para renderização gráfica simples, tratamento de eventos de teclado (`keydown`), lógica de matrizes bidimensionais para representação de mapas e o uso de métodos de array modernos como `flat()`, `filter()` e `map()` para processar e exibir dados de forma eficiente.
