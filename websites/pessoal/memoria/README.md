# Jogo da Memória

📌 **Descrição**
Um jogo de memória clássico onde o objetivo é encontrar todos os pares de números ocultos no tabuleiro.

🚀 **Tecnologias**
- HTML
- CSS
- JavaScript

🎯 **Funcionalidades**
- Embaralhamento aleatório dos itens a cada nova partida.
- Sistema de revelação de cartas ao clicar.
- Lógica de comparação para verificar se os pares são iguais.
- Ocultação automática de cartas em caso de erro.

🖥️ **Demonstração**
Link do projeto online: https://clovisbalreira.github.io/portfolio/websites/pessoal/memoria/index.html
