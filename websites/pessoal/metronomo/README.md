# Metrônomo Musical

📌 **Descrição**
Uma ferramenta digital indispensável para músicos, que auxilia na manutenção do tempo rítmico através de sinais visuais e sonoros personalizáveis.

🚀 **Tecnologias**
- HTML
- CSS
- JavaScript

🎯 **Funcionalidades**
- Ajuste preciso de BPM (Batidas Por Minuto) via controle deslizante.
- Seleção de diferentes figuras musicais (Semibreve, Mínima, Semínima, Colcheia, Semicolcheia).
- Suporte a múltiplos compassos (4/4, 3/4, 6/8).
- Feedback visual sincronizado e sons distintos para a primeira batida do compasso.
- Função "Tap Tempo" para definir o ritmo manualmente através de cliques.
- Contador de batidas e compassos em tempo real.

📚 **Aprendizados**
O desenvolvimento deste projeto envolveu o uso da API de Áudio do HTML5 para reprodução de sons, controle de tempo rigoroso com `setInterval`, lógica matemática para cálculo de intervalos e médias (especialmente na função Tap Tempo), e manipulação dinâmica de elementos do DOM e classes CSS para criar uma interface interativa e responsiva.
