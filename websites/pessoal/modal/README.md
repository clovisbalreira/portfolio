# Modal

📌 **Descrição**
Uma demonstração técnica simples e eficiente da implementação de janelas modais utilizando a tag nativa `<dialog>` do HTML5.

🚀 **Tecnologias**
- HTML
- CSS
- JavaScript

🎯 **Funcionalidades**
- Abertura de modal com efeito de backdrop (fundo escurecido).
- Fechamento intuitivo através de botão dedicado.
- Utilização de APIs nativas do navegador para gerenciamento de modais.

🖥️ **Demonstração**
Link do projeto online: https://clovisbalreira.github.io/portfolio/websites/pessoal/modal/index.html

📚 **Aprendizados**
O projeto destaca o uso de elementos modernos do HTML5 como o `<dialog>`, simplificando a criação de modais que antes exigiam bibliotecas externas ou lógica complexa de CSS/JS, além de demonstrar a estilização do pseudo-elemento `::backdrop`.
