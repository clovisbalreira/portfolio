# Nave Animada (Rocket Launch)

📌 Descrição
Um jogo estratégico de lançamento de foguete onde o jogador deve distribuir pontos entre Combustível, Propulsão e Estrutura para tentar alcançar a maior altitude possível (objetivo: 10.000 metros).

🚀 Tecnologias
- HTML
- JavaScript
- CSS

🎯 Funcionalidades
- Sistema de distribuição de pontos limitada (máximo 1000 pontos totais)
- Simulação de lançamento com física de propulsão e perda de combustível
- Cronômetro e monitoramento de altitude máxima alcançada
- Feedback visual do estado do foguete (ligado, desligado ou destruído)

📚 Aprendizados
O foco técnico deste projeto foi o gerenciamento de estado e validação de inputs do usuário. Aprendi a lidar com a proporção visual no DOM (mapeando a altitude real para pixels na tela) e a implementar condições de parada baseadas em variáveis de física e integridade estrutural do foguete.
