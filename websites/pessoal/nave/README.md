# Simulador de Nave

📌 Descrição
Um simulador minimalista de pouso lunar onde o objetivo é controlar o motor da nave para garantir um pouso suave, evitando que a velocidade de impacto seja muito alta.

🚀 Tecnologias
- HTML
- JavaScript
- CSS

🎯 Funcionalidades
- Controle de propulsão via clique do mouse (ligar/desligar motor)
- Simulação física de gravidade, aceleração e velocidade
- Monitoramento em tempo real de combustível e velocidade de descida
- Detecção automática de pouso bem-sucedido ou explosão da nave

📚 Aprendizados
Este projeto permitiu o estudo de ciclos de atualização (game loops) utilizando `setInterval`. Foi implementada uma lógica de física básica para calcular o deslocamento vertical baseado na aceleração resultante (motor vs gravidade), além de trabalhar com a manipulação direta de propriedades CSS para animar a posição da nave no cenário.
