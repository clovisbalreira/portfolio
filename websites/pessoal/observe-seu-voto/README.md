# Observe seu Voto

📌 Descrição
Portal educativo e informativo desenvolvido para conscientizar cidadãos sobre a importância do voto e facilitar o acompanhamento da atividade política no Brasil, com foco em deputados, partidos e legislação.

🚀 Tecnologias
- HTML
- CSS
- JavaScript

🎯 Funcionalidades
- Consulta detalhada de informações sobre Deputados
- Filtros de visualização por Partidos e Estados
- Cruzamento de dados entre Partidos-Estados e Estados-Partidos
- Seção informativa sobre Leis e o funcionamento do sistema eleitoral
- Interface organizada para navegação intuitiva entre diferentes categorias políticas
