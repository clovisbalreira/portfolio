# Projetos JavaScript

📌 Descrição
Uma coletânea de pequenos projetos e experimentos desenvolvidos para praticar conceitos fundamentais e avançados de lógica de programação com JavaScript.

🚀 Tecnologias
- HTML
- CSS
- JavaScript

🎯 Funcionalidades
A coleção inclui os seguintes utilitários:
- **Carro Flex**: Calculadora para decidir entre Álcool ou Gasolina.
- **Cartas**: Sorteador aleatório de cartas de baralho.
- **Dados**: Simulador de lançamento de dados.
- **Explosão**: Exemplo de animação e manipulação de eventos.
- **Lambada**: Exercício clássico de interruptor (on/off).
- **Pedra-Papel-Tesoura**: Jogo clássico contra o computador.
- **Slide**: Implementação de um slideshow/carrossel de imagens.
- **Tabuada**: Gerador automático de tabuadas.

📚 Aprendizados
Esta coleção serviu como um laboratório para diversas técnicas: manipulação do DOM, tratamento de eventos, cálculos matemáticos, lógica de jogos e controle de animações simples. A organização de múltiplos projetos em um único repositório ajudou a entender a modularidade e a reutilização de componentes básicos de UI.
