# Calculadora Flex

📌 Descrição
Calculadora de combustível desenvolvida para auxiliar motoristas na escolha entre Etanol e Gasolina, baseada no custo-benefício.

🚀 Tecnologias
- HTML
- CSS
- JavaScript

🎯 Funcionalidades
- Entrada de preços para Etanol e Gasolina.
- Cálculo automático da vantagem econômica.
- Feedback visual com alteração de status e imagens.

🖥️ Demonstração
