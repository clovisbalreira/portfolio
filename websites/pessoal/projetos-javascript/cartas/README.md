# Cartas

📌 Descrição
Simulador de sorteio de cartas de um baralho, exibindo aleatoriamente naipes e valores.

🚀 Tecnologias
- HTML
- CSS
- JavaScript

🎯 Funcionalidades
- Sorteio aleatório de naipes (copas, espadas, ouros, paus).
- Exibição dinâmica do valor da carta sorteada.
- Interface visual para visualização da carta.

🖥️ Demonstração
Link do projeto online: https://clovisbalreira.github.io/portfolio/websites/pessoal/projetos-javascript/cartas/index.html
