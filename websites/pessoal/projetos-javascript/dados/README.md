# Jogo de dado

📌 Descrição
Simulador simples de lançamento de dado de 6 faces.

🚀 Tecnologias
- HTML
- CSS
- JavaScript

🎯 Funcionalidades
- Geração de número aleatório entre 1 e 6.
- Atualização visual da face do dado conforme o resultado do sorteio.
- Interface intuitiva com apenas um clique.

🖥️ Demonstração
Link do projeto online: https://clovisbalreira.github.io/portfolio/websites/pessoal/projetos-javascript/dados/index.html
