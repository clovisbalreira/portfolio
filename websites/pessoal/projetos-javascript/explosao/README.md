# Explosão

📌 Descrição
Projeto interativo que simula uma contagem regressiva ou gatilho para um efeito visual de explosão.

🚀 Tecnologias
- HTML
- CSS
- JavaScript

🎯 Funcionalidades
- Gatilho de evento ao clicar em um elemento específico.
- Efeito visual de explosão (Xeque-Mate).
- Feedback textual durante o processo.

🖥️ Demonstração
Link do projeto online: https://clovisbalreira.github.io/portfolio/websites/pessoal/projetos-javascript/explosao/index.html
