# Lambada

📌 Descrição
Simulador interativo de uma lâmpada que pode ser ligada, desligada ou colocada para piscar.

🚀 Tecnologias
- HTML
- CSS
- JavaScript

🎯 Funcionalidades
- Botão "ON" para ligar a lâmpada.
- Botão "OFF" para desligar a lâmpada.
- Função "BLINK" que faz a lâmpada piscar repetidamente.
