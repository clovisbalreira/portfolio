# Pedra, Papel e Tesoura

📌 Descrição
O clássico jogo Pedra, Papel e Tesoura jogado contra a inteligência artificial do computador.

🚀 Tecnologias
- HTML
- CSS
- JavaScript

🎯 Funcionalidades
- Escolha da jogada do usuário através de opções (radio buttons).
- Sorteio randômico da jogada do computador.
- Comparação de resultados e exibição do vencedor com placar.

🖥️ Demonstração
