# Slide

📌 Descrição
Um carrossel de imagens (slideshow) automático para exibição de banners ou fotos.

🚀 Tecnologias
- HTML
- CSS
- JavaScript

🎯 Funcionalidades
- Troca automática de imagens após o carregamento da página.
- Exibição de banners de forma sequencial.
- Início automático via evento onload.
