# Tabuadas

📌 Descrição
Gerador automático de tabuadas matemáticas para auxiliar no aprendizado e consulta rápida.

🚀 Tecnologias
- HTML
- CSS
- JavaScript

🎯 Funcionalidades
- Cálculo instantâneo da tabuada de diferentes números.
- Exibição formatada dos resultados na tela.
- Interface simples e direta para visualização.

🖥️ Demonstração
Link do projeto online: https://clovisbalreira.github.io/portfolio/websites/pessoal/projetos-javascript/tabuada/index.html

📚 Aprendizados
Neste projeto, pratiquei o uso de laços de repetição (loops) para processamento de dados e a criação dinâmica de conteúdo textual no HTML através de concatenação de strings no JavaScript.
