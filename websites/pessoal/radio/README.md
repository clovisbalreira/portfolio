# Radio Online Portal

📌 Descrição
Um portal completo de web rádio que oferece streaming de áudio ao vivo, visualização de programação, informações sobre locutores e ferramentas de interação com os ouvintes.

🚀 Tecnologias
- HTML
- CSS
- JavaScript (Módulos/ES6)
- FontAwesome (Ícones)

🎯 Funcionalidades
- Player de áudio integrado com controles de reprodução
- Exibição dinâmica da programação da rádio e perfis de locutores
- Sistema de pedidos de música e formulário de contato
- Seção de posts e notícias integradas
- Integração com redes sociais e links externos
- Navegação fluida entre seções (SPA style)

📚 Aprendizados
O desenvolvimento deste projeto focou na criação de uma interface rica e dinâmica utilizando Módulos JavaScript para organizar a lógica de diferentes seções (player, programação, formulários). Foi um exercício excelente em manipulação assíncrona do DOM e gerenciamento de recursos de mídia (áudio).
