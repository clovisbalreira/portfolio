# Ranking C.B.F.

📌 Descrição
Sistema de visualização e consulta de dados do Ranking Nacional de Clubes da CBF, permitindo explorar a pontuação e posição de times brasileiros ao longo dos anos.

🚀 Tecnologias
- HTML
- CSS
- JavaScript (Módulos)

🎯 Funcionalidades
- Consulta de ranking por ano específico
- Filtragem de desempenho por Clube
- Agrupamento e visualização de dados por Estado
- Interface limpa focada em tabelas e dados estatísticos
- Créditos e links para a fonte oficial dos dados (CBF)

📚 Aprendizados
Neste projeto, o desafio técnico foi o tratamento e a apresentação de grandes conjuntos de dados de forma legível. O uso de módulos JavaScript permitiu separar a lógica de processamento de dados (filtragem por estado/clube) da camada de apresentação, facilitando a manutenção do código.
