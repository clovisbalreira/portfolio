# Restaurante Mais Tempero - home

📌 Descrição
Site institucional para o Restaurante Japonês Mais Tempero, apresentando a culinária clássica japonesa e o ambiente do restaurante em Canoas.

🚀 Tecnologias
- HTML
- CSS
- JavaScript

🎯 Funcionalidades
- Apresentação da culinária clássica japonesa (sushi, sashimi, tempurá).
- Informações sobre o ambiente e atendimento diferenciado.
- Estrutura de navegação para diferentes seções.

🖥️ Demonstração
Link do projeto online: https://clovisbalreira.github.io/portfolio/websites/pessoal/restaurante-japones/index.html

Neste projeto, aprendi a estruturar conteúdos textuais de forma semântica em HTML, a organizar estilos CSS para uma apresentação limpa e a utilizar scripts JavaScript para gerenciar a navegação e elementos da interface.
