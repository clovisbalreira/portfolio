# RPG Engine (Text-Based)

📌 Descrição
Um motor de jogo de RPG baseado em texto que permite aos usuários criarem seus próprios mundos através de coordenadas e navegarem por eles em uma interface minimalista.

🚀 Tecnologias
- HTML
- JavaScript

🎯 Funcionalidades
- Sistema de cadastro de cenários por coordenadas (Norte/Sul, Leste/Oeste)
- Ferramenta de Exportação e Importação de mundos via formato JSON
- Motor de movimentação direcional (Norte, Sul, Leste, Oeste)
- Renderização dinâmica de textos baseada na localização atual do jogador
- Validação de caminhos existentes (impede movimento para áreas não cadastradas)

▶️ Como rodar
1. Clone o repositório
2. Abra o arquivo index.html
3. Use a aba "Cadastrar" para criar cenários ou "Exportar/Importar" para carregar um mundo pronto.

📚 Aprendizados
Este projeto explorou o uso de objetos literais como um banco de dados em memória. O principal aprendizado foi o gerenciamento de estado e coordenadas geográficas em uma aplicação web, além de praticar a persistência de dados do lado do cliente através da manipulação de strings JSON para salvar e carregar o progresso do "construtor de mundos".
