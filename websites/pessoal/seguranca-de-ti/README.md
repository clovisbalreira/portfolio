# Segurança de T.I. Informativo

📌 Descrição
Um portal educativo dedicado aos conceitos fundamentais de Segurança da Informação e Tecnologia da Informação, abordando desde riscos na internet até controles físicos e lógicos.

🚀 Tecnologias
- HTML
- CSS

🎯 Funcionalidades
- Navegação entre capítulos especializados: Segurança de Rede, Vírus, Protocolos, etc.
- Seções dedicadas a Controles Físicos e Lógicos de segurança
- Guia informativo sobre Antivírus e proteção contra malwares
- Interface temática com iconografia relacionada à cibersegurança
- Estrutura clara e organizada para facilitar o estudo de conceitos de TI

📚 Aprendizados
O desenvolvimento deste site focou na arquitetura de informação e na organização de conteúdos técnicos de forma hierárquica. Foi um exercício prático em estruturação semântica de múltiplas páginas interconectadas e na aplicação de estilos CSS para criar uma identidade visual coerente em todo o portal educativo.
