# Animated Skills Bars

📌 Descrição
Uma interface interativa que exibe habilidades técnicas através de barras de progresso animadas, proporcionando uma forma visual e moderna de apresentar um portfólio de competências.

🚀 Tecnologias
- HTML
- CSS (Animações/Keyframes)
- JavaScript

🎯 Funcionalidades
- Barras de habilidades animadas para HTML, CSS, JS, PHP e MySQL
- Tooltips dinâmicas que exibem a porcentagem de domínio sobre cada tecnologia
- Animações de preenchimento (loading) suaves ao carregar a página
- Layout responsivo e centralizado
- Design limpo com foco em legibilidade e UX

📚 Aprendizados
Este projeto serviu para aprofundar o conhecimento em animações CSS avançadas, utilizando `@keyframes` e transições para criar efeitos de movimento. Além disso, foram praticados conceitos de posicionamento (relative/absolute) para a criação das tooltips e a manipulação de classes via JavaScript para disparar as animações de interface.
