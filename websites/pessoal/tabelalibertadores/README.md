# Tabela Libertadores & Sul-Americana 2022

📌 Descrição
Um dashboard esportivo completo para acompanhar o desenrolar da Copa Libertadores e da Copa Sul-Americana de 2022, apresentando desde a fase de grupos até o chaveamento do mata-mata e estatísticas técnicas.

🚀 Tecnologias
- HTML
- CSS
- JavaScript

🎯 Funcionalidades
- Visualização das tabelas de classificação de todos os grupos (A-H)
- Chaveamento dinâmico para as fases eliminatórias (Quartas, Semi e Final)
- Estatísticas detalhadas por técnicos e associações esportivas
- Listagem de jogos e resultados atualizados
- Interface organizada para exibição de múltiplos conjuntos de dados simultaneamente

📚 Aprendizados
Este projeto foi um excelente exercício em renderização dinâmica de dados. O foco principal foi transformar objetos JavaScript complexos em tabelas e componentes visuais (como o chaveamento de mata-mata) de forma automatizada, além de praticar a organização de CSS para layouts que contém grande volume de informação tabular.
