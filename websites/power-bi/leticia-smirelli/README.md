# Power Bi - Leticia Smirelli

📌 Descrição
Projeto de dashboard Power BI imerso em uma página web com navegação entre relatórios embutidos.

🚀 Tecnologias
- HTML
- CSS
- JavaScript
- Power BI Embedded (via iframe)

🎯 Funcionalidades
- Navegação entre diferentes relatórios Power BI: Farmácia
- Visualização de dashboards embutidos em iframe
- Mudança dinâmica de tema conforme o relatório selecionado
- Links externos para currículo, LinkedIn e GitHub

🖥️ Demonstração
Abra `index.html` no navegador para ver a interface com navegação e relatórios embutidos do Power BI.

▶️ Como rodar
1. Clone o repositório
2. Abra `index.html` no navegador

📚 Aprendizados
Implementação de uma interface MVC simples em JavaScript e integração de conteúdo externo via iframe, com navegação dinâmica e alteração de estilo baseada no relatório selecionado.
