# JavaScript - Aprendizado

📌 Descrição
Um projeto prático focado no aprendizado das bases do JavaScript, focado na organização e listagem de conteúdos educativos.

🚀 Tecnologias
- HTML
- CSS
- JavaScript

🎯 Funcionalidades
- Listagem dinâmica de aulas através de scripts JS.
- Layout centralizado e limpo para melhor foco no conteúdo.
- Estrutura modular com separação de responsabilidades (HTML/CSS/JS).

🖥️ Demonstração

📚 Aprendizados
Neste projeto, aprendi a integrar arquivos JavaScript externos para manipular elementos do DOM, como a criação de listas dinâmicas, além de reforçar conceitos de estilização básica para containers.
