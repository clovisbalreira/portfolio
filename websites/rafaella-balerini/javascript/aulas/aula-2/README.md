# Lista de Tarefas

📌 Descrição
Um aplicativo web simples para criar e gerenciar listas de tarefas, permitindo a adição dinâmica de novos itens.

🚀 Tecnologias
- HTML
- CSS
- JavaScript

🎯 Funcionalidades
- Adição de novas tarefas à lista.
- Interface simples para gerenciamento de afazeres.
- Interatividade com o usuário via prompts ou inputs.

🖥️ Demonstração
