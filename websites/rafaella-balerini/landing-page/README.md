# Balle Bot Landing Page

📌 Descrição
Uma landing page moderna desenvolvida para apresentar o "Balle Bot", um assistente de Discord especializado em segurança e gestão de comunidades.

🚀 Tecnologias
- HTML
- CSS
- JavaScript

🎯 Funcionalidades
- Menu de navegação responsivo com links para Comunidade e Tutoriais.
- Seção principal (Hero) com botão de chamada para ação (CTA).
- Listagem detalhada das funcionalidades de segurança (Captcha e Banimento Automático).

🖥️ Demonstração
