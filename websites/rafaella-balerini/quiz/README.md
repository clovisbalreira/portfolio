# Quiz para Programadores

📌 Descrição
Um web app de quiz interativo que desafia o conhecimento dos desenvolvedores em diversas áreas da tecnologia, desde HTML até lógica de programação.

🚀 Tecnologias
- HTML
- CSS
- JavaScript

🎯 Funcionalidades
- Variedade de tipos de entrada: múltipla escolha, campos de texto, datas e upload de arquivos.
- Sistema de pontuação dinâmica com tabela de avaliação de desempenho.
- Formulário de feedback integrado com resposta visual personalizada ao usuário.

📚 Aprendizados
Este projeto permitiu um mergulho profundo na manipulação do DOM e eventos de formulário. Aprendi a gerenciar múltiplos tipos de inputs, validar dados inseridos e criar uma experiência de usuário rica com feedback imediato.
