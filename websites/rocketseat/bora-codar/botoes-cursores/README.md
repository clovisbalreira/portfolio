# Galeria de Botões e Cursores

📌 **Descrição**
Showcase interativo de diferentes estilos de botões (primário, secundário, terciário) e estados de interação como loading.

🚀 **Tecnologias**
- HTML
- CSS

🎯 **Funcionalidades**
- Diferentes estilos de botões
- Estados de interação (hover, active, loading)
- Showcase de cursores

🖥️ **Demonstração**
