# Cartão de Embarque

📌 **Descrição**
Gerador de cartões de embarque digitais que exibe informações dinâmicas de voo, passageiro e segurança.

🚀 **Tecnologias**
- HTML
- CSS

🎯 **Funcionalidades**
- Layout fiel a cartões de embarque reais
- Design responsivo
- Estilização moderna

🖥️ **Demonstração**
