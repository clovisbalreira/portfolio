# Weather App UI

📌 **Descrição**
Dashboard de previsão do tempo com interface rica em detalhes meteorológicos e ícones dinâmicos.

🚀 **Tecnologias**
- HTML
- CSS
- JavaScript

🎯 **Funcionalidades**
- Previsão do tempo detalhada
- Ícones dinâmicos
- Layout moderno
