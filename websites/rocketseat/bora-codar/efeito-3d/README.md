# Efeito 3D Atropos

📌 **Descrição**
Demonstração de efeitos de profundidade e paralaxe interativos usando a biblioteca Atropos.

🚀 **Tecnologias**
- HTML
- CSS
- JavaScript
- Atropos.js

🎯 **Funcionalidades**
- Efeito de profundidade 3D
- Paralaxe ao mover o mouse
- Interface interativa

🖥️ **Demonstração**
