# Encontre seu Bloco

📌 **Descrição**
Plataforma de busca e localização de blocos de carnaval com filtros inteligentes.

🚀 **Tecnologias**
- HTML
- CSS
- JavaScript

🎯 **Funcionalidades**
- Busca de blocos
- Filtros por categoria e localização
- Listagem dinâmica

🖥️ **Demonstração**
