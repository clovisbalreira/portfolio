# Galeria de Fotos

📌 **Descrição**
Visualizador de imagens com grid responsivo e transições suaves entre fotos.

🚀 **Tecnologias**
- HTML
- CSS
- JavaScript

🎯 **Funcionalidades**
- Grid de imagens responsivo
- Visualização em tela cheia (lightbox)
- Transições suaves

🖥️ **Demonstração**
