# Kanban Board

📌 **Descrição**
Gerenciador de tarefas visual utilizando o método Kanban para organização de fluxo de trabalho.

🚀 **Tecnologias**
- HTML
- CSS
- JavaScript

🎯 **Funcionalidades**
- Colunas de status (To Do, Doing, Done)
- Arraste e solte tarefas
- Criação de novas tarefas

🖥️ **Demonstração**
Link do projeto online: https://clovisbalreira.github.io/portfolio/websites/rocketseat/bora-codar/kanban/index.html
