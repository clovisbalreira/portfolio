# Music Player Compact

📌 **Descrição**
Player de música minimalista com controles essenciais e visual moderno.

🚀 **Tecnologias**
- HTML
- CSS
- JavaScript

🎯 **Funcionalidades**
- Controles de reprodução (Play/Pause/Skip)
- Barra de progresso
- Capa do álbum

🖥️ **Demonstração**
