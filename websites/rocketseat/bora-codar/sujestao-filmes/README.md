📌 **Descrição**
Sistema interativo que recomenda filmes com base em preferências do usuário.

🚀 **Tecnologias**
- HTML
- CSS
- JavaScript

🎯 **Funcionalidades**
- Algoritmo simples de recomendação
- Listagem de filmes
- Interface de feedback
