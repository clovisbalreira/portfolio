# Gerenciador de Inscrições

📌 **Descrição**
Sistema para controle e gestão de participantes em eventos de grande escala.

🚀 **Tecnologias**
- HTML
- CSS
- JavaScript

🎯 **Funcionalidades**
- Listagem de inscritos
- Busca por participante
- Status de confirmação

🖥️ **Demonstração**
