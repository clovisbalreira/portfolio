# Gerenciador de Atividades

📌 **Descrição**
Ferramenta de produtividade para organização e acompanhamento de tarefas.

🚀 **Tecnologias**
- HTML
- CSS
- JavaScript

🎯 **Funcionalidades**
- Criação e remoção de tarefas
- Marcar tarefas como concluídas
- Filtros de status

🖥️ **Demonstração**
